# Content pipeline

## source truth
Actual course materials define:
- topics
- terminology
- notation
- formulas
- graph conventions
- methods
- exam expectations

## pipeline
1. ingest source materials
2. create source inventory
3. map sources into canonical course map
4. create source-faithful learning objects
5. label each object with source status
6. render into learn, practice, exam, and review modes
7. log learner interaction and mistakes

## source statuses
- direct-source
- source-distilled
- platform-added-explanation
- platform-added-drill
- cross-link

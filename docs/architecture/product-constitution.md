# Product constitution

## product identity
This project is a source-faithful, interactive, exam-focused university learning platform.

It is not:
- a generic study website
- a decorative course summary
- a static archive of lecture notes
- a quiz app pretending to be an exam system

It must combine:
1. source-faithful course reconstruction
2. elite pedagogy
3. interactive reasoning tools
4. performance training loops

## closed learning loop
teach -> force recall -> diagnose weakness -> adapt practice -> simulate exam pressure -> review mistakes -> resurface later

## benchmark clarification
mikro1 is the current strongest module and serves as the initial benchmark for depth, interactivity, and pedagogy.
It is not frozen and may also be improved.

## platform rule
Consistency must come from shared architecture, quality standards, learner-state logic, source traceability, and assessment design.
Consistency must not come from stripping strong module-specific features.
